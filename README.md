# XmlProcessor
Aplicação para leitura de arquivos XML de notas fiscais eletrônicas (NF-e), extraindo e exibindo impostos e informações fiscais de forma clara e automatizada.

Projeto desenvolvido para realizar a leitura e interpretação de arquivos XML de Notas Fiscais Eletrônicas (NF-e), extraindo automaticamente os impostos incidentes, como ICMS, PIS e COFINS, facilitando a análise fiscal e a conferência tributária.
